Remmina Plugin RustDesk

Introduction

View project on GitHub

Description

Remmina Plugin RustDesk is a free software protocol plugin to establish connections using RustDesk, a free open source remote control application with many features and capable of connecting even through NAT, both for Windows, Linux and others operating systems.

RustDesk installation is required in order to let this plugin to work properly.

Settings

The Remmina Plugin RustDesk plugin settings are documented in the Documentation page.

Installation

The stable versions released with source code and prebuilt packages are linked in the Download page. For information about installation from packages or source code please refer to the Installation page.

The Remmina Plugin RustDesk development version source code is available on GitHub.
