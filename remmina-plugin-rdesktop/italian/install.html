Remmina Plugin RDesktop

Installazione

Vedi il progetto su GitHub

Installazione su Debian, Ubuntu e derivate

E' possibile scaricare il pacchetto deb dalla pagina Scarica e installarlo utilizzando: 

cd "~/cartella con il file scaricato"
sudo dpkg -i "nome del file.deb"

Installazione su Arch Linux e derivate

In AUR è presente l'ultima versione disponibile, è possibile scaricarla e compilarla utilizzando yaourt (o un analogo gestore per AUR) con yaourt -S remmina-plugin-rdesktop e seguire le istruzioni del gestore di AUR.

E' anche possibile scaricare il pacchetto tar.xz dalla pagina Scarica e installarlo utilizzando: 

cd "~/cartella con il file scaricato"
sudo pacman -U "remmina-plugin-rdesktop-versione.tar.xz"

Installazione dal codice sorgente

Per compilare Remmina Plugin RDesktop dal codice sorgente è necessario avere anche Remmina Plugin Builder, un ambiente di sviluppo minimale per la compilazione di plugin per Remmina.

Scaricare Remmina Plugin Builder dalla pagina del progetto e fare riferimento alla <a href=../../remmina-plugin-builder/italian/install>pagina Installazione</a>.

Il processo di compilazione richiede semplicemente l'estrazione del codice sorgente sia di Remmina Plugin Builder sia di Remmina Plugin RDesktop utilizzando: 

cd "~/cartella con il file scaricato"
tar xzf remmina-plugin-builder-*.tar.gz
tar xzf remmina-plugin-rdesktop-*.tar.gz

Quindi sarà necessario compilare il plugin con: 

mkdir build
cp -r remmina-plugin-builder-*/* build
cp -r remmina-plugin-rdesktop-*/* build/remmina-plugin-to-build
cd build
cmake -DCMAKE_INSTALL_PREFIX=/usr .
make
sudo make install
sudo gtk-update-icon-cache -q /usr/share/icons/hicolor/
cd ..

Al termine dell'installazione, se non sono stati riportati errori sarà possibile eliminare la cartella build da root oppure tramite sudo rm -rf "build" e il file sorgente scaricato.

Dopo aver eseguito la procedura di installazione manuale da codice sorgente non sarà possibile disinstallare automaticamente il software ma sarà necessario eliminare manualmente tutti i files installati che sono indicati durante l'ultimo passaggio del processo di installazione.
