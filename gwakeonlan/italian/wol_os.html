gWakeOnLan

Abilitare il WOL nel sistema operativo

Guarda il progetto suGitHub

Introduzione

Dopo aver abilitato il Wake on LAN nel BIOS il sistema operativo deve essere informato di abilitarlo prima dello spegnimento altrimenti la macchina non si riaccenderà anche se tutto il resto è stato fatto nella maniera corretta.

GNU/Linux

Per controllare se la scheda di rete supporta il Wake On LAN il comando ethtool può essere utilizzato:

# ethtool eth0
Settings for eth0:
        ....
        Supports Wake-on: pumbg
        Wake-on: d
        .....

Se il campo Supports Wake-on contiene g allora il Wake On LAN con Magic Packet può essere utilizzato:

# ethtool -s eth0 wol g
# ethtool eth0
Settings for eth0:
        ....
        Wake-on: g
        .....

Il comando deve essere eseguito prima dello spegnimento della macchina.

Apple OS X

Nei sistemi Apple OS X l'impostazione del Wake On LAN si trova nelle Preferenze di sistema e quindi nella sezione Risparmio energia.

Microsoft Windows

Nei principali sistemi operativi Microsoft Windows il Wake On LAN è abilitato in maniera predefinita quindi in linea generale non è necessario eseguire nessuna operazione. Se la macchina ancora non si accende allora può essere necessario se il sistema operativo ha il Wake On LAN attivato col Magic Packet.

Le specifiche seguenti potrebbero essere differenti in quanto dipendenti dal driver in uso.
