Installazione

La procedura di installazione di Zabbix è semplice e possibile in numerose maniere, in base al tipo di installazione che si desidera seguire.

Installazione appliance da file ISO

Installare Zabbix Appliance da file ISO significa installare un nuovo sistema operativo, basato su Ubuntu Server, già pronto e configurato per accogliere Zabbix Server.

Installazione appliance su macchina virtuale

Installare Zabbix Appliance su macchina virtuale è possibile utilizzando un modello già pronto da scaricare e semplicemente importare nel virtualizzatore preferito.

Installazione su Arch Linux

L'installazione di Zabbix Server su Arch Linux richiede un minimo di configurazione delle varie componenti. Questa guida spiega i passaggi fino all'avvio di Zabbix Frontend Web.

Configurazione iniziale di Zabbix Frontend Web

La configurazione iniziale di Zabbix Frontend Web richiede l'inserimento dei dati di accesso del server di database e di Zabbix Server.

Conoscere Zabbix Frontend Web

Impariamo a conoscere Zabbix Frontend Web, la parte centrale per la configurazione del sistema, degli host, degli avvisi e dei valori da monitorare.

Prima di passare alla configurazione effettiva del primo sistema controllato da Zabbix è necessario conoscere il principio di funzionamento dei protocolli di monitoraggio, dei gruppi di host e degli host.

Zabbix Agent

L'agente di Zabbix è un software da installare sul sistema operativo da controllare che permette ai server di effettuare monitoraggio periodico e ricevere gli avvisi in caso di problemi riscontrati sul sistema operativo controllato.

La procedura di installazione di Zabbix Agent dipende dal sistema operativo in uso. In questa guida saranno mostrate le procedure di installazione dell'agente in vari sistemi operativi.
